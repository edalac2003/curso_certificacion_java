2. ¿Cuál de los siguientes métodos "main", tiene la sintaxis correcta?

A. Public static void main (String[ ] args) { }
B. public Static void Main (String[ ] args) { }
C. public static void main (String( ) args) { }

A. int count = 5; quantity = 2;
B. string name, label;				X
C. boolean complete = "false";
D. boolean complete = true;			X

4. Dado que:

String name = "Bob";
String  msg;
int num = 3;

¿Cuál de las siguientes sentencias asigna el valor de "Bob wrote 3 Java programs." a la variable msg?

A. msg = name + " wrote " + num " Java programs.";
B.  msg = name + " wrote " + 3 + " Java programs.";			X
C.  msg = name + " wrote " + (2 + 1) + " Java programs.";	X
D. msg = name + " wrote " + 2 + 1 + " Java programs.";
